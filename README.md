# LoopLens

> Gain Clear Insights into Player Behavior

## Overview

**Problem:** Indie game developers not knowing why players quit their games.

**Solution:** Captures game sessions and provides insights into player behavior.

- **Backend**: Node.js + Express + tRPC
- **Database**: PostgreSQL + Drizzle ORM
